# 🧙 La Machine à Invocation

## 📖 Description

La Machine à Invocation est une petite application web réalisée dans le cadre d'un kata JavaScript.

L'utilisateur saisit son prénom, choisit une créature fantastique, puis clique sur **Invoquer**. L'application affiche alors un message personnalisé et masque le formulaire.

---

## ✨ Fonctionnalités

- Saisie d'un prénom.
- Sélection d'une créature fantastique.
- Génération d'un message personnalisé.
- Masquage du formulaire après validation.
- Affichage du résultat.
- Interface responsive.
- Thème fantasy personnalisé.

---
